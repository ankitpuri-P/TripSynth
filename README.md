# ⚡ TripSynth: AI Travel Concierge

TripSynth is a fully interactive, AI-powered travel agent built with Python, LangChain, and Streamlit. It synthesizes custom, day-by-day travel itineraries and automatically generates interactive Google Maps links for every location.

### 🚀 [Click Here to Try the Live App!](https://tripsynth-mnemefsgopjcv4qvhwsn3h.streamlit.app/)

## Features
* **Lightning Fast Generation:** Powered by Groq's LLaMA 3 70B model.
* **Live Web Search:** Uses DuckDuckGo to pull up-to-date information.
* **Interactive UI:** Custom CSS styling with a sleek Tron-glow aesthetic.
* **Smart Mapping:** Automatically generates Universal Google Maps routing for every recommended location.
